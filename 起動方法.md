# リズトレカレンダー 起動方法

**Cursor 内のターミナルではポートが使えないため、サーバーが起動しません。**

## 手順（必ずこちらで起動してください）

1. **Macの「ターミナル」を開く**
   - Spotlight（⌘+Space）で「ターミナル」と入力して起動
   - または アプリケーション → ユーティリティ → ターミナル

2. **次のコマンドを順に実行**
   ```bash
   cd /Users/fukitakatsumi/RZT-SCH-APP
   npm run dev
   ```

3. **表示されたURLをブラウザで開く**
   - 例: `http://localhost:3001/`
   - ターミナルに表示される「Local:」のURLをコピーしてブラウザに貼り付けてください。

4. **終了するとき**
   - ターミナルで `Ctrl + C` を押す

---

※ 3001が他で使われている場合は、Viteが自動で別のポート（3002など）を表示します。そのURLを開いてください。
